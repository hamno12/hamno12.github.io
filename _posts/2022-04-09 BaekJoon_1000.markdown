#  A+B
## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.



This is code
```java
import java.util.*;
public class BJ1000 {
    public static void main(String[] args) {
        Scanner scanner =new Scanner(System.in);
        int A = scanner.nextInt();
        int B = scanner.nextInt();
        int sum = 0;
        if (A <= 0){
            return;
        }
        if (B >= 10){
            return;
        }
        sum = A + B ;
        System.out.println(sum);
    }
}
```
